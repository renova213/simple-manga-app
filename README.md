MangaTime is comic reader apps that provides comics in bahasa Indonesia, using firebase for backend.

## Features

- [x] List of Category
- [x] Read chapter vertically or horizontally
- [x] Favorite Comic
- [x] Search Comic by name and Genres
- [x] Report Form
